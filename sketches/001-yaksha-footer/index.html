Yaksha FAQ — Footer Design Variants

Issue 6: Header + footer consistency

Goal: Make the footer consistent with the Navbar — same brand identity, navigation, and visual weight. The current footer is too minimal ("Missiles FAQ Portal" was the old wrong name). Three design directions below — pick one and I'll apply it.

Variant A — Compact row

A: Compact row

Single-row layout, brand left, nav center, social right. Clean and space-efficient. Best for pages where footer doesn't compete for attention.

Variant B — Two-row split

B: Two-row split

Top row has brand + link columns. Bottom row has copyright and social icons. More informative, gives the footer real presence. Best for content-heavy pages.

Variant C — Centered minimal

C: Centered minimal

Centered, breathing layout. Brand above, nav, divider, social, copyright stacked. Calm and elegant — matches the editorial feel of the FAQ site. Best for content-first pages.
